快捷键

2020-03-29

文本操作（通用）：
1.Ctrl+C = 复制
2.Ctrl+V = 粘贴
3.Ctrl+A = 全选
4.Ctrl+X = 剪切
5.Ctrl+Z = 撤回
6.Ctrl+Shift+R = 选中整块
7.End = 光标跳到行尾
8.Home = 光标跳到行首
9.Shift+Home = 选中从当前光标位置到这一行的行首
10.Shift+End = 选中从当前光标位置到这一行的行尾

Windows常用快捷键：
1.F3 = 打开资源管理器搜索结果
2.F11 = 最大化文件显示/恢复正常显示
3.Win+E = 打开"我的电脑"
4.Win+W = 关闭当前窗口
5.Win+R = 打开运行窗口
6.Win+D = 最小化所有窗口，显示桌面
7.Win+L = 锁定计算机，回到登录窗口
8.Win+M = 最小化当前窗口
9.Win+U = 打开设置
10.Win+Home = 将正在使用之外的所有其他窗口最小化
11.Win+break = 打开控制面板系统属性
12.Win+↑/↓/←/→ = 窗口位置
13.Ctrl+F = 查找（各种编辑器也通用）
14.Ctrl+D = 将选定项目至回收站
15.Ctrl+Shift+Esc = 打开资源管理器搜索当前页面
16.Ctrl+Shift+N = 新建文件
17.Alt+Tab = 切换至上个窗口
18.Alt+Enter = 打开选中项属性
19.Alt+F4 = 关闭当前程序
